** Rules for creating files in this directory **

 - All files in this directory MUST be markdown 
   - Best practice is to create complete, actionable prompts suitable for ingestion by an LLM based coding agent 
     - With markdown checkboxes for discrete sub-tasks allowing for incremental progress over multiple turns
     - Human targeted justification for the work including the potential cost of not completing it
     - Code snippets, ascii art diagrams, mermaid diagrams, or anything else which clarifies either the problem or intended solution
