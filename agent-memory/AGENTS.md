*Coding agents shared memory*
- This directory is a place to keep notes about tasks being worked on and track progress within a single task. 
- It can also be used for debugging sessions which require multiple tasks to complete. 
- Space in this directory is limited, you are welcome to use it but do not waste it.


** Rules for creating files in this directory **

 - All files in this directory should be markdown 
 - If you need to put a crashdump, logfile or other artifact of debugging into "memory" for a subsequent run it must  be accompanied by a MANIFEST_filename.md 
    - The MANIFEST identifies the contents and their purpose and at least one other markdown "thought" file which refers to this "memory"

*ONLY HUMANS DELETE FILES*

- If an agent is done with thoughts or memories they should be renamed accordingly as part of the commit which obsoletes them and a human will later review and delete as appropriate
